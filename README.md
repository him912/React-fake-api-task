# React Fake Store API Task

A modern React e-commerce application built with **Vite**, **React Router**, and **Axios** that displays products from the [Fake Store API](https://fakestoreapi.com/).

## 📋 Features

- ✨ **Product Listing** - Browse products from Fake Store API
- 🔍 **Product Details** - View detailed information in modal popup
- 🛒 **Shopping Cart** - Add products to cart and manage quantities
- ➕➖ **Quantity Control** - Increment/decrement item quantities easily
- 📊 **Order Summary** - See cart totals and item breakdown
- 🎯 **Responsive Design** - Clean, user-friendly interface
- ⚡ **Fast Performance** - Built with Vite for optimal speed

## 🛠️ Tech Stack

- **React 19** - UI library
- **React Router DOM 6** - Client-side routing
- **Vite 8** - Build tool
- **Axios** - HTTP client for API calls
- **CSS3** - Styling
